  Overview
</h2>

**CodeQuest Lite** is your personal coding challenge tracker designed to help you master algorithms and data structures. Whether you're preparing for technical interviews, participating in competitive programming, or simply improving your problem-solving skills, CodeQuest Lite provides the tools you need to organize your practice and visualize your progress.

  Installation
</h2>

<div align="center">
  
  ```zsh
  # Clone the repository
  git clone https://github.com/yourusername/codequest-lite.git
  
  # Navigate to the project directory
  cd codequest-lite
  
  # Install dependencies
  npm install
  
  # Start the development server
  npm start
```
